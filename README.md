# Rainbow Six Siege Strat Guide Editor

🛠️ Hobby project to create an online tool for drawing custom map strategies for Rainbow Six Siege.

## Features so far

✅ Upload and display static map blueprint images  
✅ Freehand drawing (Pen tool) — draw lines on the map  
✅ Select tool — select individual drawn lines  
✅ Delete selected line  
✅ Undo last line  
✅ Clear all lines (with confirmation)  
✅ Sidebar with tabs (Operators / Utility — content TBD)  
✅ Notes section for strategy notes  
✅ Responsive UI — modern toolbar and layout

## Tech stack

- React
- React-Konva (for canvas and drawing)
- Lucide React (icons)
- Google Fonts: Saira Condensed
